# Match-3

Небольшая браузерная игра в жанре match-3, собранная на `Phaser 3` и `TypeScript`.

В проекте уже реализованы:
- игровое поле с настраиваемым размером;
- swap соседних фишек;
- поиск матчей и каскады;
- счёт и рестарт;
- звуковые эффекты и фоновая музыка;
- деплой на GitHub Pages.

## Технологии

- `TypeScript`
- `Phaser 3`
- `Vite`
- `ESLint`
- `Prettier`

## Запуск локально

Установка зависимостей:

```bash
npm install
```

Запуск dev-сервера:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

Предпросмотр production-сборки:

```bash
npm run preview
```

## О проекте

Проект сделан как небольшой playable-прототип match-3 без серверной части. Основной фокус здесь на базовом игровом цикле, визуале, ощущении от свайпа и матчей, а также на том, чтобы в игру можно было сразу зайти в браузере и поиграть.

## Vibe Coding

Игра завайбкожена при помощи `Codex` в CLI-режиме: значительная часть итераций, логики, UI-правок и интеграции фич делалась через работу с Codex в терминале.
